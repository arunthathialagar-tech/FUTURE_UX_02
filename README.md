# 📅 Appointment Booking App UI

A modern and user-friendly Appointment Booking App UI designed in **Figma**. The application provides an intuitive interface for users to schedule appointments, view bookings, and manage their profiles seamlessly.

## ✨ Features

* Clean and modern UI design
* Appointment scheduling interface
* User-friendly navigation
* Responsive layout
* Professional color scheme
* Easy booking experience

## 🛠 Tools Used

* **Figma** – UI/UX Design
* **GitHub** – Project Repository
